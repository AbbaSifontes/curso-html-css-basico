Texto

El texto es el contenido principal de una página web. HTML ofrece diferentes elementos para estructurar y dar formato al texto.

Negrita: se utiliza para resaltar palabras o frases importantes. Se puede lograr con la etiqueta <strong> que le da un sentido semántico texto importante o con la etiqueta <b> texto en negrita pero sin sentido semántico.

Cursiva: se utiliza para enfatizar palabras o frases. Se puede lograr con la etiqueta <em> o con la etiqueta <i> ejemplo: texto en cursiva.

Salto de línea : se utiliza <br> para crear un salto de línea sin iniciar un nuevo párrafo. Es un elemento vacío, lo que significa que no tiene una etiqueta de cierre.

Línea horizontal : se utiliza <hr> para crear una línea horizontal que divide secciones de contenido. Es un elemento vacío. Ejemplo:

Código: se utiliza para mostrar código fuente o fragmentos de código. Se puede lograr con la etiqueta <code>. Ejemplo: console.log("Hola Mundo");.

Pre-formateado: se utiliza para mostrar texto con un formato predefinido, como el código fuente o el arte ASCII. Se puede lograr con la etiqueta <pre>. Ejemplo: 

           Texto preformateado
           con salto de linea
           y espacios-intencionales.

Subrayado: se utiliza para resaltar palabras o frases. Se puede lograr con la etiqueta <u>.

Tachado: se utiliza para indicar que un texto ha sido eliminado o no es relevante. Se puede lograr con la etiqueta <del>.

Enlaces

Los enlaces son elementos que permiten navegar entre diferentes páginas web o secciones dentro de la misma página.

Para crear un enlace externo, se utiliza la etiqueta <a> con el atributo href que especifica la URL de destino,resultado visual: <a href="https://www.google.com" target="_blank"> Ejemplo: Enlace a Google.

También se pueden crear enlaces internos que navegan a secciones dentro de la misma página utilizando el atributo href con un identificador de sección,resultado visual: <a href="#id-seccion"> Ejemplo: Ir a la sección de texto.

Enlaces de Correos se usan para crear enlaces que abren el cliente de correo predeterminado del usuario. resultado visual: <a href="mailto:ejemplo@dominio.com"> Ejemplo:Enviar correo

Imágenes

Las imágenes son elementos que permiten mostrar contenido visual en una página web.

Para insertar una imagen se utiliza la etiqueta <img> con el atributo src que especifica la URL de la imagen y el atributo alt que proporciona una descripción alternativa de la imagen para mejorar la accesibilidad,los atributos width y height se usan para definir el tamaño de la imagen se pueden definir inline pero lo recomendado es usar CSS. Ejemplo: <img src="ruta-de-la-imagen.jpg" alt="Descripción de la imagen" width="250" height="250"> resultado visual:

otra forma de insertar una imagen es usando la etiqueta <figure> que se utiliza para agrupar una imagen y su leyenda, y la etiqueta <figcaption> que se utiliza para proporcionar una leyenda o descripción de la imagen. Ejemplo: <figure> <img src="ruta-de-la-imagen.jpg" alt="Descripción de la imagen"> <figcaption>Leyenda de la imagen</figcaption>

Formularios

Los formularios son elementos que permiten a los usuarios enviar información a un servidor web.

Para crear un formulario se utiliza la etiqueta <form> con el atributo action que especifica la URL a la que se enviará el formulario y el atributo method que especifica el método HTTP (GET o POST) que se utilizará para enviar el formulario ejemplo: <form action="procesar-formulario.php" method="POST"> . Dentro del formulario se pueden incluir diferentes tipos de campos de entrada, como <input>, <textarea>, <select>, entre otros. Ejemplo:

Multimedia

Los elementos multimedia permiten incorporar contenido audiovisual en una página web.

Para insertar un video se utiliza la etiqueta <video> con el atributo src que especifica la URL del video, el atributo controls para mostrar los controles de reproducción y el atributo preload para precargar el video automáticamente. Ejemplo: <video src="ruta-del-video.mp4" controls preload>.

Para insertar un audio se utiliza la etiqueta <audio> con el atributo src que especifica la URL del audio, el atributo controls para mostrar los controles de reproducción y el atributo preload para precargar el audio automáticamente. Ejemplo: <audio src="ruta-del-audio.mp3" controls preload>.

Para insertar un iframe se utiliza la etiqueta <iframe> con el atributo src que especifica la URL del contenido a mostrar, el atributo width para definir el ancho del iframe y el atributo height para definir la altura del iframe. Ejemplo: <iframe src="https://www.youtube.com/embed/dQw4w9WgXcQ" width="560" height="315">. Ejemplo de insercion de un video de youtube usando iframe:

Explicacion de ¿Qué es un iframe?

Un iframe (inline frame) es un elemento que permite insertar otro documento HTML dentro de la página actual.

Los iframes se utilizan comúnmente para mostrar contenido de terceros, como videos de YouTube, mapas de Google Maps o widgets de redes sociales, sin necesidad de redirigir al usuario a otra página.

El contenido dentro del iframe se carga de forma independiente al contenido de la página.
